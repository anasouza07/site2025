Modelo RGB

O modelo RGB (Red, Green, Blue) é um modelo aditivo de cores, ou seja, as cores são criadas através da combinação de luzes vermelha, verde e azul.

Funcionamento do Pixel no Modelo RGB

O pixel é a menor unidade de uma imagem digital e, no modelo RGB, ele é composto por três subpixels (vermelho, verde e azul). A intensidade de cada cor é combinada para gerar a cor final.

Modelo CMYK

O modelo CMYK (Cyan, Magenta, Yellow, Key/Black) é um modelo subtrativo de cores, onde as cores são criadas a partir da combinação de ciano, magenta, amarelo e preto.

Comparação: RGB x CMYK

O modelo RGB é utilizado em dispositivos de luz, como monitores e TVs, enquanto o CMYK é utilizado em impressão. A principal diferença é que RGB é aditivo e cria cores com luz, enquanto CMYK é subtrativo e trabalha com pigmentos.

  • RGB: Red, Green, Blue (aditivo).
  • CMYK: Cyan, Magenta, Yellow, Key (subtrativo).
