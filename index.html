Modelos de Cor RGB e CMYK

Explicação do modelo RGB

O modelo RGB (Red, Green, Blue) é um sistema aditivo de cores utilizado em dispositivos como monitores e televisores, onde a luz vermelha (R), verde (G) e azul (B) são combinadas em diferentes intensidades para formar diversas cores.

Funcionamento do Pixel no RGB

Em um sistema RGB, cada pixel de uma tela é composto por três sub-pixels: um vermelho, um verde e um azul. A intensidade de cada cor varia de 0 a 255, criando milhões de combinações de cores.

Explicação do modelo CMYK

O modelo CMYK (Cyan, Magenta, Yellow, Key/Black) é um modelo subtrativo usado principalmente para impressão. Nele, as cores primárias ciano (C), magenta (M) e amarelo (Y) são misturadas para criar outras cores, e o preto (K) é adicionado para proporcionar sombras e profundidade.

Comparação RGB x CMYK

Enquanto o RGB é baseado na adição de luz para gerar cores, o CMYK funciona pela subtração de luz, o que é mais adequado para a impressão. RGB é utilizado em telas e CMYK é usado na impressão de materiais gráficos.
